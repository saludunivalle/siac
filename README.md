
# SIAC

**SIAC** (Sistema Integral de Aseguramiento de la Calidad) es una plataforma web desarrollada para la gestión, seguimiento y visualización de indicadores relacionados con la calidad académica de los programas de la Facultad de Salud de la Universidad del Valle.

## ¿Para qué sirve?

El sistema permite:

- Visualizar el estado de acreditación y registro calificado de los programas académicos.
- Hacer seguimiento a procesos de alta calidad y renovación de acreditaciones.
- Consultar estadísticas y consolidar información histórica de los programas.
- Gestionar y monitorear indicadores clave de docencia, servicio y otros aspectos relevantes.
- Facilitar la toma de decisiones mediante dashboards y reportes visuales.

## ¿Cómo funciona?

SIAC cuenta con una interfaz web desarrollada en React y un backend en Node.js/Express. Utiliza Google Sheets como base de datos para almacenar y consultar información, integrando autenticación mediante Google para acceso seguro.

El sistema está orientado a usuarios administrativos, directivos y personal encargado del aseguramiento de la calidad en la Facultad de Salud.
